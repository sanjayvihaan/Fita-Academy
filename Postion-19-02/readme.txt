In CSS, the static position is the default positioning value for an element. 
It means that the element is rendered in the normal document flow, 
and its position is not affected by other positioning properties like top, right, bottom, or left.

Here is an example of using the static position in CSS:
div.static {
    position: static;
    border: 2px solid #000;
}
